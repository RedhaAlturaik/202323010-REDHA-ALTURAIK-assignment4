# AI Usage Report

## Tools Used and Use Cases

### ChatGPT

ChatGPT was used as a development support tool during this assignment. I used it to:

- brainstorm ways to transform the earlier portfolio into a more polished final web application
- review and improve JavaScript logic for filtering, sorting, local storage, and modal behavior
- refine UI and UX decisions such as section structure, navigation flow, and feedback messaging
- help strengthen documentation for the README and technical report
- identify improvements that would better match the Assignment 4 rubric

## Benefits and Challenges

### Benefits

Using AI helped me:

- move faster when planning feature improvements
- compare different implementation ideas before choosing one
- understand cleaner ways to organize JavaScript functions
- improve the professionalism of the final documentation and UI content

### Challenges

Using AI also had limitations:

- some suggestions were too generic and needed adaptation to fit my project
- generated code sometimes required cleanup or restructuring to stay readable
- not every design idea fit the assignment scope, so I had to choose carefully
- AI suggestions still required testing and verification before being trusted

## Learning Outcomes

By using AI during this assignment, I improved my understanding of:

- how to organize a larger front-end project more clearly
- how to create better user feedback and graceful fallbacks
- how to combine static portfolio content with dynamic API-based content
- how to evaluate AI suggestions instead of accepting them blindly

## Responsible Use and Modifications

I used AI responsibly by treating it as a support tool rather than an automatic solution. To maintain correctness, originality, and academic integrity:

- I reviewed all AI-generated suggestions before applying them
- I edited and rewrote suggestions to match my own project structure and style
- I tested the final behavior manually
- I kept only the ideas and code that I understood and could explain
- I documented the use of AI transparently in this report and in the README

The final website reflects my own decisions, modifications, and understanding of the implementation.
